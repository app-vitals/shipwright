Comparison

A self-hosted AI coding agent — entirely in your cloud.

"Self-hosted" gets used loosely. The real question is architectural: does the whole system — control plane included — run on infrastructure you own, or does part of it always phone home to someone else's cloud? Here is the honest, sourced breakdown.

Facts verified as of July 14, 2026, from each vendor's own primary sources (linked below). Re-checked immediately before this page ships — vendor terms and tiers change.

Fully in your cloud, or hybrid?

Shipwright's control plane, agent runtime, and your code all run on infrastructure you own, at every tier — no managed-service layer in between. The rest of the field is either hybrid (part of the system stays in the vendor's cloud) or gates full self-hosting behind an Enterprise sales conversation.

Tool License Architecture
Shipwright MIT Fully in your own cloud — control plane included. The admin, metrics, and agent runtime all deploy to infrastructure you own (Docker or your own Kubernetes cluster). No managed service sits between your repository and the agent.
Cursor (self-hosted cloud agents) Proprietary Hybrid — inference and orchestration stay in Cursor's cloud. Only tool execution and artifacts run on your infrastructure. No spec phase, no deploy stage. [source]
OpenHands MIT root, time-limited PolyForm carve-out on enterprise/ Productized VPC self-host is Enterprise-gated. The MIT license applies to the local tier; VPC self-hosting for the delivery-pipeline features is sold, not shipped free — the vendor's own site states "Talk to sales" for it. [source]
Coder Agents Governance platform, not an OSS agent No built-in cloud runtime to compare — it wraps agents you run. Scope stops at code, tests, and PRs. No spec phase, no deploy stage, no built-in cron. [source]
Devin Proprietary Cognition-hosted at every tier. Even the dedicated single-tenant VPC option is a Cognition-managed environment — never inside your own infrastructure. Full breakdown at /vs/devin. [source]

"Self-hosted" is table-stakes marketing language across the field now — the architecture underneath it is what actually differs. Full head to head on Devin specifically: /vs/devin →.

Economics

Shipwright: free, MIT, no tiers or seats — the whole pipeline, not a gated tier of it. Everyone else in the table above: see each vendor's own site for current terms, which change over time.
